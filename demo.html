Installation

Add the widget to your website with just a single script tag:

<!-- Add to your HTML -->
<script src="https://yourcdn.com/feedback.js"></script>
<script>
  Feedback.init({
    apiUrl: 'https://your-api.com/feedback',
    projectId: 'your-project-id'
  });
</script>

Or use query parameters for quick setup:

<script src="https://yourcdn.com/feedback.js?project=PROJECT_ID&api=API_URL"></script>

Build from Source

To build the widget from source code:

# Install dependencies
npm install

# Build for production
npm run build

# Development mode with watch
npm run dev

# Start development server
npm start
